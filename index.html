Desarrollo web frontendUDEMY ACADEMY

En este curso aprendí las herramientas necesarias para crear la estructura de una pagina web y como subirla a internet

UX DesignerUDEMY ACADEMY

En el curso de UI/UX (Interfaz de Usuario/Experiencia de Usuario) aprendí los principios fundamentales para diseñar interfaces de usuario efectivas y atractivas. Aprendí sobre la importancia de comprender a los usuarios y sus necesidades, así como a realizar investigaciones de usuarios y pruebas de usabilidad. Además, aprendí a crear wireframes y prototipos interactivos, a utilizar herramientas y técnicas de diseño visual, y a aplicar los principios de diseño de interacción para mejorar la experiencia del usuario en diferentes plataformas y dispositivos.

Bases de DatosUDEMY ACADEMY

En el curso de bases de datos MySQL aprendí los fundamentos de este sistema de gestión de bases de datos relacionales. Aprendí a diseñar y crear bases de datos utilizando el lenguaje SQL (Structured Query Language) y a manejar tablas, consultas y relaciones entre los datos. También aprendí a realizar operaciones de inserción, actualización, eliminación y consulta de datos utilizando sentencias SQL. Además, adquirí conocimientos sobre la optimización y el rendimiento de las consultas, así como sobre la seguridad y la administración de bases de datos MySQL.

BOOTSTRAP UDEMY ACADEMY

En el curso de Bootstrap aprendí a utilizar este popular framework de desarrollo front-end. Aprendí a crear sitios web responsivos y atractivos utilizando las clases y componentes predefinidos de Bootstrap. También aprendí a personalizar y adaptar estos componentes según mis necesidades, y a utilizar las funcionalidades de Bootstrap para mejorar la usabilidad y la experiencia del usuario en mis proyectos.

ANGULARUDEMY ACADEMY

En el curso de Angular aprendí a utilizar este framework de desarrollo front-end para crear aplicaciones web escalables y de alto rendimiento. Aprendí los conceptos básicos de Angular, como los componentes, módulos y servicios. También aprendí a utilizar el enlace de datos bidireccional, las directivas y los enrutadores de Angular para crear aplicaciones dinámicas. Además, adquirí conocimientos sobre cómo trabajar con APIs y cómo implementar características avanzadas, como la autenticación de usuarios y la gestión del estado de la aplicación.

Github UDEMY ACADEMY

En el curso de GitHub aprendí a utilizar esta plataforma de desarrollo colaborativo y control de versiones. Aprendí a crear y gestionar repositorios, a clonar y bifurcar proyectos existentes, y a colaborar con otros desarrolladores mediante solicitudes de extracción y comentarios. También aprendí a utilizar el sistema de control de versiones de Git para realizar seguimiento de cambios en el código, crear ramas y fusionar cambios. Además, adquirí conocimientos sobre las características adicionales de GitHub, como el seguimiento de problemas y la integración con servicios de implementación continua.
